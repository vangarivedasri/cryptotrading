\# Simplified Binance Futures Trading Bot

\*\*Author:\*\* VedaSri Vangari  

\*\*Language:\*\* Python  

\*\*Libraries:\*\* python-binance, tkinter, logging  



---



\##  Description

This project is a simplified trading bot that interacts with the \*\*Binance Futures Testnet\*\*.  

It supports both \*\*command-line\*\* and \*\*graphical (Tkinter)\*\* interfaces.



---



\##  Features

1. &nbsp;Connects to Binance Futures Testnet  
2. &nbsp;Market, Limit, and Stop-Market Orders  
3. &nbsp;Supports Buy/Sell sides  
4. &nbsp;Input validation  
5. &nbsp;Logging and error handling  
6. &nbsp;Simple Tkinter GUI  
7. &nbsp;User confirmation before placing orders
